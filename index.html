Fantasy::IRC

A modular, event-driven IRC client/bot gem with plugin support for Ruby

Installation

Add this line to your application's Gemfile:

gem 'fantasy-irc'

And then execute:

$ bundle

Or install it yourself as:

$ gem install fantasy-irc
